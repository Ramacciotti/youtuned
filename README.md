# YouTuned

YouTuned é uma extensão para personalizar a experiência do YouTube. Ela remove blocos de Shorts, restaura o contador de dislikes quando possível, e permite esconder vídeos e comentários com base em palavras bloqueadas.

## O que a extensão faz

- Remove automaticamente seções de Shorts da interface do YouTube.
- Oculta vídeos e comentários que contenham palavras específicas informadas pelo usuário.
- Restaura o contador de dislikes em páginas de vídeo, quando a API pública estiver disponível.
