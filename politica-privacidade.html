Política de Privacidade

Última atualização: 17/07/2026

Esta Política de Privacidade descreve como a extensão "Youtuned" (doravante "Extensão", "nós", "nosso") trata as informações dos usuários.

1. Coleta e Uso de Dados

A Extensão NÃO coleta, armazena ou compartilha nenhum dado pessoal identificável dos seus usuários.

  • Dados de Navegação: A Extensão não rastreia, registra ou armazena seu histórico de navegação.
  • Dados Pessoais: Não solicitamos nem temos acesso a informações como nome, e-mail, endereço ou localização.
  • Palavras Bloqueadas: As palavras e frases que você insere para bloqueio são armazenadas exclusivamente no seu navegador usando a API `storage` do Chrome. Isso permite que suas preferências sejam salvas entre as sessões. Estes dados não são enviados para nós ou para terceiros.

2. API do Return YouTube Dislike

Para restaurar a contagem de "não gostei" (dislikes) no YouTube, a Extensão faz requisições à API pública do Return YouTube Dislike (returnyoutubedislikeapi.com).

  • Dados Enviados: A única informação enviada para esta API é o ID do vídeo do YouTube que você está visualizando no momento.
  • Finalidade: Este dado é utilizado exclusivamente para buscar a contagem de dislikes daquele vídeo específico e exibi-lo na interface do YouTube para você.
  • Não Compartilhamento: Nenhum dado pessoal seu (como IP ou informações de conta) é enviado para esta API. Para mais detalhes sobre o tratamento de dados pela API, consulte a política de privacidade e documentação do Return YouTube Dislike [citation:6].

3. Dados de Uso e Hospedagem

Esta página de política de privacidade é hospedada em uma plataforma de terceiros (GitHub Pages). Como qualquer provedor de hospedagem, eles podem registrar informações de acesso padrão (como endereço IP) para fins de segurança e operação. Recomendamos que você consulte a política de privacidade do hospedeiro para mais detalhes [citation:2].

4. Alterações nesta Política

Podemos atualizar esta Política de Privacidade ocasionalmente. Quaisquer alterações serão publicadas nesta página.
